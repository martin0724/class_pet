# 课堂宠物养成游戏策划案

## 1. 游戏概述

### 1.1 游戏名称
《小伙伴》（ClassPet）

### 1.2 游戏类型
教育类宠物养成游戏（网页端）

### 1.3 游戏定位
一款面向小学生的课堂激励机制，通过宠物养成系统激励学生积极参与课堂问答，培养学生的责任感、爱心和团队协作精神。

### 1.4 核心玩法
学生选择并养成自己的虚拟宠物，通过课堂积极回答问题获得互动机会，使用互动机会与宠物进行喂养、游戏等互动，提升宠物成长阶段和各项属性。

### 1.5 目标用户
- 主要用户：小学生（6-12岁）
- 次要用户：授课教师

### 1.6 使用场景
课堂教学实时使用，支持多人同时在线游玩。

---

## 2. 游戏背景

在神奇的"宠物星球"上，每一位小朋友都有一个专属的宠物蛋。这些宠物蛋需要小朋友的悉心照料才能孵化、成长。小朋友在课堂上越积极主动，宠物就获得越多能量，与主人的感情就越深厚，最终成长为最亲密的小伙伴。

## 3. 宠物系统

### 3.1 宠物列表

| 序号 | 名称 | 类型 | 性格特点 |
|------|------|------|----------|
| 1 | 小汪 | 狗狗 | 忠诚友好，喜欢互动 |
| 2 | 喵喵 | 猫咪 | 高冷傲娇，需要耐心 |
| 3 | 咕咕 | 小鸡 | 活泼可爱，容易满足 |
| 4 | 跳跳 | 小兔 | 蹦蹦跳跳，精力旺盛 |
| 5 | 毛毛 | 小羊 | 温柔乖巧，善于撒娇 |
| 6 | 牙牙 | 小马 | 勇敢热情，喜欢奔跑 |
| 7 | 蛋蛋 | 恐龙 | 神秘稀有，成长惊喜大 |

### 3.2 宠物外观风格
- 整体风格：Q版卡通
- 形象特点：圆润的身体、大眼睛、简化的四肢
- 尺寸比例：头大身小，2:1或3:1的头身比
- 配色：明亮鲜艳，适合儿童审美
- 动画：待机时有呼吸/眨眼等微动作

### 3.3 宠物属性系统

| 属性名称 | 属性说明 | 数值范围 | 状态影响 |
|----------|----------|----------|----------|
| 饱食度 | 宠物饥饿程度 | 0-100 | 喂养可增加，低于20时宠物会不开心 |
| 健康值 | 宠物的身体健康状态 | 0-100 | 游戏互动可增加，低于20时宠物无精打采 |
| 心情值 | 宠物的情绪状态 | 0-100 | 陪伴互动可增加，影响其他属性恢复速度 |
| 亲密度 | 宠物与主人的感情 | 0-100 | 长期互动积累，影响宠物成长速度 |
| 经验值 | 宠物当前升级进度 | 0-100 | 通过各类互动获得 |

### 3.4 宠物状态阈值

| 状态 | 数值范围 | 视觉表现 |
|------|----------|----------|
| 极佳 | 80-100 | 宠物非常开心，有特殊动画 |
| 良好 | 60-79 | 宠物状态正常 |
| 一般 | 40-59 | 宠物略显疲惫 |
| 低迷 | 20-39 | 宠物无精打采，趴下 |
| 危险 | 1-19 | 宠物虚弱倒地 |
| 死亡 | 0 | 宠物消失，显示墓碑图标 |

### 3.5 宠物死亡与复活机制
- 当任意属性降至0时，宠物进入"死亡"状态
- 死亡后宠物从主界面消失，显示墓碑图标
- 复活条件：由老师根据学生课堂表现决定是否复活
- 复活方式：老师点击复活按钮，宠物恢复至初始状态
- 复活惩罚：宠物成长阶段下降一级（若有）

## 4. 成长系统

### 4.1 成长阶段

| 阶段 | 名称 | 所需经验 | 解锁内容 | 外观变化 |
|------|------|----------|----------|----------|
| Lv1 | 蛋 | 0 | 初始状态 | 宠物蛋形态 |
| Lv2 | 幼儿 | 100 | 孵化完成 | 幼崽形态，体型小巧 |
| Lv3 | 青年 | 300 | 解锁宠物比赛 | 体型增大，开始有特征 |
| Lv4 | 成年 | 600 | 解锁特殊技能 | 完全体，细节丰富 |
| Lv5 | 冠军 | 1000 | 解锁时装系统 | 发光特效，稀有外观 |

### 4.2 经验值获取

| 行为 | 经验值 | 说明 |
|------|--------|------|
| 喂养 | +5 | 使用互动机会喂养宠物 |
| 游戏 | +8 | 与宠物玩游戏 |
| 清理 | +3 | 清理宠物房间 |
| 比赛获胜 | +15 | 宠物对战获胜 |
| 比赛参与 | +5 | 宠物参与比赛 |
| 连续互动3次/天 | +10 | 每日活跃奖励 |

### 4.3 成长规则
- 宠物达到100经验值时自动升级
- 升级时播放庆祝动画和音效
- 每升一级解锁新的互动类型或外观
- 成年后可参与宠物比赛

## 5. 互动系统

### 5.1 互动机会机制

**获取方式：**
- 课堂回答问题正确：获得1次互动机会
- 连续回答正确3次：额外获得1次互动机会（连击奖励）
- 老师可根据题目难度手动奖励1-3次机会

**机会记录：**
- 互动机会显示在宠物界面右上角
- 格式：🔔 剩余次数：X
- 机会可累积，无上限

**使用规则：**
- 学生自主选择使用时机
- 使用后次数-1
- 不可转让给他人

### 5.2 互动类型

| 互动类型 | 消耗次数 | 饱食度 | 健康值 | 心情值 | 亲密度 | 经验值 |
|----------|----------|--------|--------|--------|--------|--------|
| 喂养食物 | 1 | +25 | - | +5 | +3 | +5 |
| 投球游戏 | 1 | - | +15 | +20 | +10 | +8 |
| 抚摸互动 | 1 | - | - | +30 | +15 | +3 |
| 清理房间 | 1 | - | +10 | +10 | +5 | +3 |
| 外出散步 | 2 | - | +20 | +25 | +20 | +8 |
| 生日派对 | 3 | +15 | +15 | +40 | +30 | +12 |

### 5.3 互动流程

```
1. 学生点击"互动"按钮
2. 显示可用互动类型列表
3. 学生选择互动类型
4. 检查互动机会是否充足
   - 若充足：播放互动动画，显示属性变化
   - 若不足：提示"互动机会不足"
5. 更新宠物属性和界面显示
```

### 5.4 被动衰减机制
为增加游戏挑战性，宠物属性会随时间自然衰减：

| 属性 | 每日衰减 | 说明 |
|------|----------|------|
| 饱食度 | -10 | 宠物会饿 |
| 健康值 | -5 | 需要运动保持健康 |
| 心情值 | -5 | 需要陪伴 |

## 6. 社交系统（宠物比赛）

### 6.1 比赛类型

| 比赛名称 | 参与条件 | 消耗次数 | 奖励 |
|----------|----------|----------|------|
| 短跑比赛 | 青年期以上 | 2次 | 经验+15，胜利称号 |
| 跳跃比赛 | 青年期以上 | 2次 | 经验+15，胜利称号 |
| 吃货大赛 | 任意阶段 | 3次 | 饱食度+50 |
| 才艺展示 | 成年期以上 | 3次 | 经验+20，解锁动作 |

### 6.2 比赛流程

```
1. 老师发起比赛
2. 学生报名（消耗对应次数）
3. 显示参赛宠物列表
4. 老师点击"开始比赛"
5. 播放随机结果动画（2-3秒）
6. 公布结果：冠军、亚军、季军
7. 获胜者获得奖励和称号
```

### 6.3 称号系统
- 短跑冠军、跳跃冠军、吃货大王、才艺之星
- 称号显示在宠物头像下方
- 可累积获得多个称号

## 9. 游戏流程

### 9.1 启动流程（老师）
```
1. 老师打开游戏网页
2. 进入老师控制面板
3. 导入或创建班级学生名单
4. 设置每节课互动机会基础值
5. 学生进入游戏选择宠物
```

### 9.2 学生首次进入流程
```
1. 输入自己的姓名
2. 从宠物列表中选择喜欢的宠物
3. 给宠物取一个昵称
4. 进入宠物养成主界面
```

### 9.3 课堂互动流程
```
老师环节：
1. 课堂提问
2. 学生回答（正确/错误）
3. 老师判定后，点击学生头像发放互动机会

学生环节：
4. 看到互动机会增加提示
5. 决定现在使用或积累
6. 进行互动操作
7. 观察宠物属性变化
```

### 9.4 每日游戏流程
```
1. 学生打开游戏（输入姓名）
2. 查看宠物状态（可能因衰减变化）
3. 使用互动机会进行互动
4. 参加老师发起的比赛
5. 关闭游戏（数据自动保存）
```

## 11. 游戏平衡性建议

### 11.1 互动次数建议
- 每节课基础发放：2-3次
- 简单回答正确：+1次
- 难题回答正确：+2次
- 连击奖励：+1次（每3连击）

### 11.2 属性衰减平衡
- 确保学生每天至少登录一次才能维持宠物健康
- 避免宠物过快死亡，增加游戏紧迫感但不至过于挫折

### 11.3 成长速度预估
- 轻度玩家：约2-3周达到成年
- 中度玩家：约1-2周达到成年
- 积极玩家：约1周达到成年
